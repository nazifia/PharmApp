- **Kling AI / Hailuo**: paste prompt as-is, set duration, generate
- **Runway Gen-3**: use Text-to-Video, add style preset "cinematic"
- **Pika Labs**: paste prompt, set aspect ratio 9:16 (mobile) or 16:9 (landscape)

## Style Consistency Tips
- Always include: `dark background, glassmorphism, teal accent, cinematic`
- Avoid: `bright lighting, white background, cartoonish`
- Aspect ratio: **16:9** for YouTube/social, **9:16** for TikTok/Reels
